# YiqiOS — 个人网站规划书 v1.0

> 一句话:把作品集做成一个"你自己的操作系统",网站本身就是第一个 vibecoding 作品。

---

## 一、核心理念

### 1.1 命题

普通人放 PDF 作品集,你放一个能玩的东西。这个网站要同时证明三件事:内容判断力、vibecoding 能力、审美。证明方式不是"说",而是让访客亲手用到。

### 1.2 世界观:YiqiOS

整个网站是一个虚构的个人操作系统。瑞士编辑网格是"桌面",作品和趣味装在可拖拽的"窗口"里。这个形式的妙处:

- 窗口是天然容器——玄学、美剧、命盘,任何怪东西装进系统窗口里都显得有秩序
- 「克制的外壳 + 有趣的内核」正是"科技感又有趣"的结构性解法
- 拖拽、开关窗口本身就是互动展示,不需要额外堆特效

### 1.3 精神主轴:维特根斯坦

不做装饰性引言,而是让哲学成为信息架构:

| 位置 | 用法 |
|---|---|
| 开机序列 | `Die Grenzen meiner Sprache bedeuten die Grenzen meiner Welt.`<br>语言的界限,即世界的界限。<br>所以我一直在扩建语言:中文 / 英文 / 代码。 |
| 各窗口副标题 | 以「语言游戏」命名规则:projects.app —"游戏规则:让它跑起来";content.app —"游戏规则:替观众说话"(呼应"意义即用法") |
| 页脚 | `"Don't think, look." — 别想,去看。`(给访客的行动指令) |

开机三行就是全站 thesis:一个学媒体的人为什么去写代码,一句话讲完。

## 二、视觉系统

### 2.1 色彩 Token

| Token | 色值 | 用途 |
|---|---|---|
| paper | `#F2F2EF` | 背景纸灰白 |
| ink | `#141414` | 正文、边框、大标题 |
| **teal(主)** | `#0FA396` | 品牌色:链接、发带、水晶坠、代码高亮、选中态 |
| red(仅错误) | `#E8390E` | Debug 模式、报错弹窗、关闭按钮 |
| mute | `#6E6E68` | 次要文字 |
| line | `#D9D9D2` | 细分隔线 |

语义纪律:**蓝绿是品牌,红色是 error**——在系统隐喻里自洽。玄学窗口、美剧窗口也不许引入新颜色。

### 2.2 字体

| 角色 | 字体 | 用法 |
|---|---|---|
| Display | Archivo(加宽轴 110%) | 超大标题,全大写,负字距 |
| Body | Inter + PingFang SC | 正文 |
| Utility | JetBrains Mono | 终端、菜单栏、标签、changelog、窗口标题栏 |

### 2.3 版式

- 12 列网格,最大宽度 1280px
- 细黑边框(1px)、零圆角、零阴影渐变
- 大留白 + 超大字号对比,瑞士海报式层级

### 2.4 个人形象:「17」小人

**风格**:扁平插画,曲线造型,黑白 + 蓝绿三色限定
**特征**:长直发中分、黑色乐队T、蓝绿水晶项链坠、腮红两粒(可爱底线,不越界)
**已定姿态库(六态)**:

| 姿态 | 触发位置 | 动画 |
|---|---|---|
| ✌️ 比耶 | Contact 按钮 | hover 挥手两下 |
| 💻 敲代码 | Projects 按钮 | hover 屏幕代码行闪烁 |
| 📣 举喇叭 | Content 按钮 | hover 冒声波线 |
| 🔍 放大镜 | Debug 开关 | 开启时举镜到眼前,眼睛变大 |
| 🏊 蛙泳 | 加载中 | 波浪线循环滚动 |
| 😴 睡觉 | 挂机 60s | 飘 z,鼠标动惊醒 |

**动效纪律**:默认静止,只在 hover/触发时动;同屏最多一个小人在动。

**配套资产(待产)**:B 方向几何徽章版"证件照"(同源五官)用于 favicon / 公众号头像 / 水印。

**补充方案**:About 区放一张真人照片的「墨黑+蓝绿」duotone 处理(瑞士编辑经典手法),卡通与真人各司其职。

## 三、结构与内容

### 3.1 开机序列(第一印象)

黑屏 → 终端逐行打字(启动日志 + 维特根斯坦三行)→ 界面"编译"展开成桌面。
`prefers-reduced-motion` 用户直接跳到桌面。

### 3.2 桌面(主界面)

顶部 mono 菜单栏:时间、状态、**五行指示器**(如 `今日:火旺 · 宜发布`,像电量图标)。
桌面图标(双击/点击打开窗口):

| 窗口 | 内容 |
|---|---|
| **projects.app** | VidScout / 妈妈的小程序(1,853 阅读故事)/ 泳池日志 |
| **content.app** | 蜂快印知识卡系统 / OiShokudo 病毒方法论 / Inkside 漂洋计划 |
| **about.txt** | 定位一句话 + duotone 照片 + 履历(赛塔林月销10万等) |
| **玄学.exe** | 八字命盘按数据表渲染:四柱网格排列、五行条形图、`// 伤官生财:靠才华和表达换钱的系统配置` |
| **tv.guide** | 「正在追」美剧节目单;双击某剧,标题栏闪一句台词彩蛋 |
| **changelog** | 剧集格式:`S2026E07 — 上线个人网站` |

### 3.3 窗口规范

细黑边框、mono 标题栏、红色关闭按钮、可拖拽、可层叠置顶。
**手机端降级**:窗口变全屏卡片(iOS App 式打开),拖拽仅桌面端。

## 附录 · v1.1 增补（2026.07.03）

### A. 方向分叉记录（重要决策点）

v1.0 定的是「纸白 + 墨黑 + 蓝绿」瑞士编辑风,但实际建成的 index.html v0.2/v0.3 走了**深水暗色玻璃风**:深水黑 `#060D0F` + 冰川乳蓝 `#8FC1BB` + 宋体哲学感 + 毛玻璃窗口。

**建议:沿深水方向走。** 理由:更贴「游泳者」世界观(整个 OS 活在水下);暗色 + 微光粒子的氛围感是纸白做不到的;在中文个人网站里辨识度更高。纸白方案降级为备选留档。原 token 表中 `#0FA396` 等色值对应替换为当前 `--teal #8FC1BB` 系。**语义纪律不变:蓝绿是品牌,红 `#E8390E` 仅 error/Debug。**
